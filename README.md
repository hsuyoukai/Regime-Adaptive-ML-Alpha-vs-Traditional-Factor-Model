# Regime-Adaptive-ML-Alpha-vs-Traditional-Factor-Model
自適應 Alpha 策略：基於 XGBoost 的多因子模型研究
本專案旨在探討機器學習（XGBoost）在選股策略中的應用，並與傳統固定權重的多因子模型進行比較 。研究顯示，機器學習能更有效地因應市場狀態（Regime Shift）的變化，並提供更穩定的超額報酬 。

## 核心目標
提升報酬率：透過機器學習優化選股，打造報酬率優於傳統模型的多因子策略 。
動態權重調整：克服傳統模型權重固定、面對市場轉折適應慢的缺點 。
捕捉非線性特徵：利用機器學習自動學習並觀測數據中的非線性特徵 。

## 模型與研究架構
### 1. 研究假說
機器學習選股策略在 IC/IR、投資報酬率與夏普值（Sharpe Ratio）等風險調整後報酬上，顯著優於傳統多因子策略 。


### 2. 模型設定與比較 
傳統模型 (Baseline)
權重配置+固定權重 
->IC 最高的前 20 個簡單平均 

XGBoost 模型
動態調整權重 +因子篩選	
超參數設定：學習率 0.05, 樹數量 300, 最大深度 3 
訓練窗口：3 年滾動窗口，Step=62


### 3. 交易規則
起始資金：100,000,000 
再平衡頻率：5 日 
選股規則：
進場：預測排名在前 5% 
出場：排名跌出前 70% 
交易成本：手續費 0.35%（含證所稅、券商手續費等） 

## 實證結果與績效 (2020 - 2025) 
在相同樣本與交易規則下，XGBoost 展現了更強的選股能力：
總報酬率：XGBoost 為 110.71%，遠優於傳統模型的 -21.84% 
夏普率 (Sharpe Ratio)：XGBoost 為 0.8202，傳統模型為 -0.0019 
最大回撤 (Max Drawdown)：XGBoost 降至 35.70%（傳統模型為 54.02%） 
IC 平均與 IR：XGBoost 分別達到 0.0989 與 0.8433 

## 關鍵洞察
市場分化是 Alpha 的來源：Alpha 的優勢取決於市場是否提供足夠的橫截面分化空間 
2023 年表現：AI 股引發市場結構性分化，個股基本面差異放大，有利於橫截面模型發揮 
2025 年挑戰：AI 主題逐漸 Beta 化，個股相關性上升，模型需更有效區分「實質獲利」與「題材驅動」的標的
